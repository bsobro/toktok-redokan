WordPress Frontend User area.

== Description ==

Allow your users to create, edit and delete posts directly from the WordPress frontend area.

= Translations =

You can [add a translation of the RH Frontend Pro](https://bitbucket.org/bizdirect/rehub-frontend-pro)
by creating a new issue where you need to point a title like 'France translation files'. Then choose the
kind of message - enhancement and upload the zip file with 2 files in it - .mo and .po

The simpliest and popular plugin for translation of WordPress files is [Loco Translate](https://wordpress.org/plugins/loco-translate/)

== Installation ==

1. Upload the entire `rh-frontend` folder to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.

You will find 'RH Frontend' menu in your WordPress admin panel.
